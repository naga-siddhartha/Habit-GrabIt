Ritual Log

Ritual Log is a habit-tracking app for Apple platforms (iOS, iPadOS; macOS coming soon). Track daily habits, build streaks, and stay consistent with a simple, focused experience.

Features

  • Create and manage habits with custom names, icons, and colors
  • Track completions and view history on a calendar
  • Streaks and statistics to keep you motivated
  • Optional reminders (local notifications)
  • Optional Sign in with Apple for iCloud sync across devices
